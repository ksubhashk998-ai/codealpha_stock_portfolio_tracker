# codealpha_stock_portfolio_tracker
📈 Task 2: Stock Portfolio Tracker

📌 Overview

The Stock Portfolio Tracker is a beginner-friendly Python project that allows users to manage and evaluate their stock investments. Users can enter stock symbols and quantities, and the program calculates the total investment value using predefined stock prices. The results can also be saved to a file for future reference.

This task is designed to help beginners practice essential Python programming concepts in a practical way.

🎯 Project Objective

Build a simple stock tracking system that calculates total investment based on:

User-entered stock symbols

User-entered quantities

Hardcoded stock prices stored in a dictionary

🛠 Key Features

📊 Hardcoded stock prices using Python dictionaries

⌨️ User input for multiple stocks and quantities

🧮 Automatic calculation of individual and total investment value

📄 Displays a clear portfolio summary

💾 Option to save results in .txt or .csv format

✅ Input validation for improved reliability

📚 Concepts Used

Dictionaries

input() / print()

Loops

Conditional statements

Arithmetic operations

File handling (.txt, .csv)

🧾 Example Stock Prices (Hardcoded)

{ "AAPL": 180,

"TSLA": 250,

"GOOGL": 140,

"MSFT": 330,

"AMZN": 135 }

▶️ How to Run

Ensure Python 3.x is installed on your system

Save the file as stock_portfolio_tracker.py

Open terminal or command prompt

Run the program using:

python stock_portfolio_tracker.py

💾 Output Files

portfolio.txt → Human-readable investment summary

portfolio.csv → Spreadsheet-friendly format for further analysis
